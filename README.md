┌─────────────────────────────────────────────────────────────┐
│ ⚙️ НАСТРОЙКА settings.json                                   │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  📌 max_pinned — сколько лотов закреплять                   │
│  💰 sell_price — цена продажи                               │
│  💰 max_price — макс. цена покупки (должна быть ниже)      │
│                                                              │
│  🔍 ПАРАМЕТРЫ ПОИСКА:                                        │
│  ├─ country: ["US"] — страна (US, RU, UA, KZ и др.)        │
│  ├─ password: "no" — без пароля / "yes" — любые            │
│  ├─ spam: "no" — без спам-блока / "yes" — любые           │
│  └─ not_origin: ["phishing"] — исключить фишинг            │
│                                                              │
│  🚫 allowed_sellers — покупать только у указанных          │
│  🚫 ignored_sellers — черный список продавцов               │
│                                                              │
