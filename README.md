## 📖 Инструкция по настройке

### 1. Установка
- Создайте папку, например `C:\LZTBooster`
- Поместите туда файл `LZTBooster.exe`
- Запустите программу

### 2. Настройка
При первом запуске программа автоматически создаст файл `settings.json` и попросит ввести API токен LZT.market.

После авторизации вы можете отредактировать `settings.json`:

- **`api_token`** — ваш токен LZT.market
- **`max_pinned`** — количество одновременно закрепленных товаров (зависит от вашего тарифа на LZT)
- **`sell_price`** — цена продажи
- **`max_price`** — максимальная цена покупки (должна быть ниже `sell_price`)
- **`check_interval`** — интервал проверки (сек)

### 3. Настройка поиска
--json
"search_params": {
    "country": ["US"],         // Страна: US, RU, UA, KZ и др.
    "password": "no",          // no - без пароля, yes - любые
    "spam": "no",              // no - без спам-блока, yes - любые
    "not_origin": ["phishing"] // Исключить фишинговые аккаунты
}
4. Настройка Telegram-бота (опционально)
Для удаленного управления создайте файл tg_bot.key в папке с программой. Формат:

text
ТОКЕН_БОТА:ПАРОЛЬ
Токен получите у @BotFather

Пароль придумайте сами (нужен для входа в Telegram-пульт)

5. Запуск
После запуска выберите в меню:

⚡ Перепродажа — полный цикл: покупка → продажа → закрепление

📌 Закрепления — только автозакрепление и поднятие лотов

❓ Частые вопросы
Бот ничего не покупает

Проверьте баланс на LZT.market

Убедитесь, что max_price не слишком низкая

Проверьте правильность API токена в settings.json

Ошибка API

Токен можно получить в настройках профиля на LZT.market

Убедитесь, что ваш тариф LZT позволяет использовать API

Как покупать только у одного продавца?

Заполните массив allowed_sellers в settings.json:

json
"allowed_sellers": ["ник_продавца"]
Как отключить черный список?

Очистите массив ignored_sellers:

json
"ignored_sellers": []
