**⚡ LZTBooster — Инструкция по настройке**

**Установка**

1. Создай любую папку на рабочем столе (например `LZTBooster`)
2. Закинь `LZTBooster.exe` внутрь папки
3. Запусти программу — она создаст `settings.json` автоматически
4. Введи API токен от LZT.market когда программа попросит
5. После первого запуска выйди из программы и открой `settings.json` для настройки

**Настройка settings.json**

`max_pinned` — сколько товаров будет закреплено одновременно. Зависит от твоего тарифа на LZT.
```json
"max_pinned": 4
```

`country` — страна аккаунтов для покупки. По умолчанию США.
```json
"country": ["US"]
```
Другие варианты: `RU`, `UA`, `KZ` и т.д.

`password` — покупать ли аккаунты с паролем двухэтапки.
```json
"password": "no"
```
`no` — только без пароля, `yes` — любые включая с паролем.

`spam` — покупать ли аккаунты со спам-блоком.
```json
"spam": "no"
```
`no` — только чистые, `yes` — любые.

`not_origin` — исключить определённые типы аккаунтов. По умолчанию исключены фишинговые.
```json
"not_origin": ["phishing"]
```

`sell_price` и `max_price` — цена продажи и максимальная цена покупки. Бот покупает дешевле и продаёт дороже.
```json
"sell_price": 30.0,
"max_price": 29.0
```
Маржа = sell_price − max_price. В примере это 1 руб с аккаунта.

`title` и `title_en` — название твоего товара на русском и английском.
```json
"title": "Телеграмм | без ограничений",
"title_en": "Telegram | no limits"
```

`description` — описание товара, пишется в одну строку.
```json
"description": "Аккаунт перепродаётся - все вопросы в лс"
```

`allowed_sellers` — если хочешь покупать только у конкретных продавцов. По умолчанию пусто = покупает у всех.
```json
"allowed_sellers": ["ник_продавца"]
```
Несколько продавцов:
```json
"allowed_sellers": ["ник1", "ник2"]
```
Чтобы покупать у всех — оставь пустым:
```json
"allowed_sellers": []
```

**Запуск**

В меню программы два режима:

**⚡ Перепродажа** — бот сам ищет аккаунты, покупает, подтверждает и выставляет на продажу с закреплением.

**📌 Бот закреплений** — только закрепляет уже купленные аккаунты, без покупки новых.

---

**Частые вопросы**

— *Бот ничего не покупает* — проверь баланс на LZT и правильность API токена в settings.json

— *Ошибка API* — токен можно получить в настройках профиля на LZT.market

— *Хочу покупать только у одного продавца* — заполни `allowed_sellers`
