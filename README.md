🚀 УСТАНОВКА
  1. Создать папку, положить LZTBooster.exe
  2. Запустить → ввести API токен LZT.market
  3. Войти по логину/паролю LOLZ

⚙️ НАСТРОЙКА (settings.json)
  max_pinned    — сколько лотов закреплять
  sell_price    — цена продажи
  max_price     — макс. цена покупки (должна быть ниже sell_price)
  country       — страна: ["US"], ["RU"], ["UA"] и т.д.
  password      — "no" (без пароля) / "yes" (любые)
  spam          — "no" (без спама) / "yes" (любые)
  allowed_sellers   — покупать только у указанных
  ignored_sellers   — черный список продавцов

  👤 Покупка у одного → allowed_sellers: ["ник_продавца"]
