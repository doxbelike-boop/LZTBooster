## 📖 Инструкция по настройке

### 1. Установка
```bash
# Создайте папку
mkdir C:\LZTBooster

# Поместите файл LZTBooster.exe в папку
# Запустите программу
C:\LZTBooster\LZTBooster.exe
```

### 2. Настройка settings.json

При первом запуске программа автоматически создаст файл `settings.json` и попросит ввести API токен LZT.market.

**Основные параметры:**

| Параметр | Описание | Пример |
|----------|----------|--------|
| `api_token` | Ваш API токен LZT.market | `"ваш_токен"` |
| `max_pinned` | Кол-во закреплений (зависит от тарифа LZT) | `3` |
| `sell_price` | Цена продажи (RUB) | `30.0` |
| `max_price` | Макс. цена покупки (RUB) | `29.0` |
| `check_interval` | Интервал проверки (сек) | `3` |

### 3. Настройка поиска

```json
"search_params": {
    "country": ["US"],           // Страна: US, RU, UA, KZ, GB и др.
    "password": "no",            // no — без пароля, yes — любые
    "spam": "no",                // no — без спам-блока, yes — любые
    "not_origin": ["phishing"]   // Исключить фишинговые аккаунты
}
```

### 4. Настройка продажи

```json
"fast_sell_params": {
    "title": "Телеграмм аккаунт",           // Название на русском
    "title_en": "Telegram account",         // Название на английском
    "description": "Аккаунт перепродается", // Описание товара
    "category_id": 24,                      // Категория Telegram
    "currency": "rub"                       // Валюта
}
```

### 5. Фильтрация продавцов

```json
// Покупать только у указанных продавцов
"allowed_sellers": ["ник_продавца1", "ник_продавца2"],

// Черный список продавцов (бот добавляет сам при ошибках)
"ignored_sellers": ["плохой_продавец"]
```

### 6. Настройка Telegram-пульта (опционально)

Создайте файл `tg_bot.key` в папке с программой:

```
ТОКЕН_БОТА:ПАРОЛЬ
```

**Как получить:**
1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получите токен: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
3. Придумайте пароль (нужен для входа в Telegram-пульт)

**Пример файла:**
```
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz:мойпароль123
```

### 7. Запуск

После запуска программы выберите в меню:

| Режим | Описание |
|-------|----------|
| **⚡ 1. Перепродажа** | Полный цикл: покупка → продажа → закрепление |
| **📌 2. Закрепления** | Только автозакрепление и поднятие лотов |

## ❓ Частые вопросы

<details>
<summary><b>❌ Бот ничего не покупает</b></summary>

- Проверьте баланс на LZT.market
- Убедитесь, что `max_price` не слишком низкая
- Проверьте API токен в `settings.json`
- Убедитесь, что есть свободные слоты для закрепления
</details>

<details>
<summary><b>🔑 Ошибка API / 401 Unauthorized</b></summary>

- Токен можно получить в настройках профиля LZT.market
- Убедитесь, что ваш тариф LZT позволяет использовать API
- Проверьте, что токен скопирован полностью (без лишних пробелов)
</details>

<details>
<summary><b>👤 Как покупать только у одного продавца?</b></summary>

Заполните массив `allowed_sellers` в `settings.json`:
```json
"allowed_sellers": ["ник_продавца"]
```
Можно указать несколько:
```json
"allowed_sellers": ["ник1", "ник2", "ник3"]
```
Чтобы покупать у всех — оставьте пустым:
```json
"allowed_sellers": []
```
</details>

<details>
<summary><b>🚫 Как добавить продавца в черный список?</b></summary>

Добавьте ник в `ignored_sellers`:
```json
"ignored_sellers": ["плохой_продавец"]
```
Бот автоматически добавляет продавцов при ошибке 403 (игнорирование)
</details>

<details>
<summary><b>📌 Почему не закрепляются лоты?</b></summary>

- Проверьте `max_pinned` — возможно, лимит исчерпан
- Убедитесь, что на балансе достаточно средств для закрепления
- Проверьте, что лоты активны и не закреплены
</details>

<details>
<summary><b>📊 Как посмотреть статистику?</b></summary>

В меню программы выберите пункт "Статистика" или в Telegram-пульте нажмите кнопку "📊 Статистика". Файл отчета сохраняется в `telegram_arbitrage_report.txt`
</details>
